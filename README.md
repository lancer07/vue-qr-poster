# create a qr-poster by Vue(二维码海报生成器)

![avatar](https://s1.ax1x.com/2020/08/28/dou7jK.png)
## example
####  import Poster from 'vue-qr-poster'
```
  <poster
    shareLink="[需要生成二维码的链接]"
    bgPic="[背景图片]"
    bgSize="573,686"
    :qrSize="200"
    qrColor="#000"
    qrBackground="rgba(255,225,255,1)"
    qrPosition="180,350"
  ></poster>
```
